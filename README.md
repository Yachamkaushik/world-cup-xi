# World Cup XI

Draft a World Cup squad one player at a time, then watch them compete through a full tournament — group stage, knockouts, and (if you're lucky) the final.

## How it works

- Spin to reveal a random national squad each round
- Pick one player for the current position from that squad
- Build a full XI across 11 rounds
- Simulate an 8-match World Cup run: 3 group games + Round of 32 → Final
- Only a small percentage of runs end in a trophy

Two modes:
- **Classic** — player stats visible, more strategic
- **Purist** — names only, pure instinct

## Stack

React 18 + Vite · Tailwind CSS · Framer Motion
